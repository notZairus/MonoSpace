# MonoSpace
## A web app that combines task, notes, and pomodoro feature in one single dashboard.

---
### Core Features:

1. **Task Management**: Add, update, delete, or complete task easily with the use of user friendly task panel.
2. **Intelligent Note-Taking:** Type notes manually or extract them automatically from files (`.pdf`, `.docx`, `.xlsx`, `.pptx`) powered by Apache Tika and Gemini.
3. **Configurable Pomodoro Timer**: Improve focus using the built-in pomodoro timer. This pomodoro timer lets user configure the duration of work, short-breaks, and long-breaks.

---
### Tech Stack

**Frontend:** 
* React & TypeScript 
* TanStack Query (State management & data fetching) 
* Tailwind CSS 

**Backend & Database:** 
* Node.js & Express 
* Prisma ORM 
* PostgreSQL 
* Apache Tika (Document text extraction) 

**DevOps / CI/CD:** 
* GitHub Actions

---
### What I learned from this project

1. **CI/CD Automation**: Configured automated CI/CD pipelines pipelines using Github Actions and Docker.
2. **Prisma ORM**: Developed a typesafe data management using prisma together with zod.
3. **Tanstack Query**: Learned how to cache and invalidate the fetched data using Tanstack Query.
